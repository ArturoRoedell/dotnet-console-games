Play from the first person perspective and shoot some baddies.

## Input

- `W`, `A`, `S`, `D`: move/look
- `Spacebar`: shoot
- `1`: equip pistol
- `2`: equip shotgun
- `M`: toggle map
- `Tab`: toggle stats
- `Escape`: exit
